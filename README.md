# xPBox 编解码应用 (Svelte + TS + Vite)

[xPBox星盒应用平台](https://www.xpbox.cn/)是一个消费级aPaaS平台，本项目是xPBox开源应用xPBox 常用编解码

## 目前支持的编解码类型

##### 编解码
- Base64编码
- Hex (utf8)
- URL编码
##### 哈希 (Hash)
- MD5
- SHA1
- SHA3
- SHA224
- SHA256
- SHA384
- SHA512
